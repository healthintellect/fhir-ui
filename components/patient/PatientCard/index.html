PatientCard

PatientCard displays a material-ui card component with the FHIR patient's data passed into it using the patient property. Different styles of fields can be used as shown below.

Properties

patient
Object
detailLink
String
fieldVariant
String

Basic usage

​
​
​
​
<Container>
  <Grid container spacing={3} direction="row">
    <Grid item xs={12}>
      <PatientCard patient={patient} fieldVariant="outlined" />
    </Grid>
  </Grid>
</Container>
